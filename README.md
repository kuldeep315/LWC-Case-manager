# LWC Case manager

A Salesforce Lightning Web Component project (SFDX) that implements a Case Manager component: search, filter, sort and paginate Cases. Built with LWC + Apex + SOQL. Ready for GitHub and SFDX.

## Features
- Search by Subject & Description
- Server-side filtering: Status, Priority, Owner, Date range
- Server-side sorting & pagination
- Returns total record count for pagination UI
- Apex implemented with attention to SOQL injection & governor limits
- Sample Apex test included

## Prerequisites
- Salesforce CLI (sfdx)
- A Salesforce Dev Hub + Scratch Org or Sandbox/Dev org
- Git
